Syntax Açıklama
$("*") Tüm elemenleri seçer
$(this) bulunduğu elementi seçer
$(p.intro) p elementi içerisindeki intro class etiketi olanları seçer
$(p:first) ilk p elementine erişilir
$(ul li:First) ul elementlerindeki ilk li ementini seçer
$("[href]") href attributene sahip elementi seçer
$("a[target!='_blank']") a elementleri içersinde target attribute _blank değerine sahip olmayanları seçer
$(":button") Tüm button elemeneleri, input içerisinde type button olan elementleri seçer
$("tr:even") tüm even tr elementleri seçer
$("tr:odd") tüm even tr elementleri seçer
